# gettingData
Getting Data Project

# Instructions

1. Save the run_analysis.R file to a folder
2. Using a text editor modify the "workingDirectory" variable at the top of the file to point to the base directory containing the unzip files of the UCI HAR Dataset. 
**Note:** this is the directory with the README.txt
3. Save the R file
4. Run it
5. The working directory will now have "tidydata.csv" created in it
